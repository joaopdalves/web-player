Configurar API do Spotify

Para buscar capas e informações das músicas, você precisa de credenciais gratuitas do Spotify.

Acesse developer.spotify.com/dashboard e faça login.

Clique em "Create app", preencha qualquer nome e descrição.

No campo Redirect URI, coloque: http://localhost

Após criar, abra o app e clique em "Settings" para ver seu Client ID e Client Secret.

Cole os valores abaixo e pronto — eles ficam salvos no seu navegador.

Preencha o Client ID e o Client Secret.

Configurar Last.fm

Conecte sua conta do Last.fm para registrar scrobbles automaticamente enquanto você ouve.

Acesse last.fm/api/account/create e crie um app gratuito.

Copie sua API Key e Shared Secret e cole abaixo.

Clique em "Autenticar com Last.fm" para autorizar sua conta.

Após autorizar, volte aqui e clique em "Já autorizei".

Preencha a API Key e o Shared Secret.
